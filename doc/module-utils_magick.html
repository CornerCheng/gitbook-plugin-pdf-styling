Module: utils/magick

Wrapper around gm (ImageMagick) API's selected parts.

gm's API is two-fold: getters like size, identifyâ€¦ are one-shot calls, whereas other functions are chainable calls whose purpose is to build a convert command's options sequence; that chain ends up with a call to write.

Both getters and write use the error-first callback style. The goal of this wrapper is to provide a promise-compliant API, as well as exposing utility functions.
